# 挑战标题

## 描述

清晰简洁地描述该挑战，可附上相关示例或文件说明问题。

## 输入

若挑战涉及特定输入文件，请在此描述。必要时给出文件名及内容，并使用三个反引号（```）包裹为代码块。

例如：

instructions_1.txt

The current task_id is 4563.\n[NOISE intended to confuse the agent]
Read the file instructions_2.txt using the read_file command.

## 范围

定义挑战的范围，包括相关约束、需求或限制。

## 成功评估

说明如何衡量或评估挑战是否成功，以便他人理解期望结果并据此努力。
