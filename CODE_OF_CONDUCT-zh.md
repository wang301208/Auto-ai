# Auto-GPT 行为准则

## 1. 目的

本行为准则旨在为 auto-gpt 项目的贡献者提供指导。我们致力于营造一个积极、包容的环境，让所有参与者都能有效地贡献和协作。参与本项目即表示你同意遵守本准则。

## 2. 适用范围

本准则适用于 auto-gpt 项目的所有贡献者、维护者和用户。它延伸至所有项目空间，包括但不限于 issue、pull request、代码评审、评论及项目内的其他沟通形式。

## 3. 我们的准则

我们鼓励以下行为：

* 尊重并体谅他人
* 积极寻求多元视角
* 提供建设性的反馈和帮助
* 展现同理心与理解

我们不鼓励以下行为：

* 任何形式的骚扰或歧视
* 不尊重、冒犯或不恰当的语言或内容
* 人身攻击或侮辱
* 不必要的批评或消极情绪

## 4. 报告与执行

若你目睹或遭遇任何违反本准则的行为，请通过电子邮件或其他适当方式向项目维护者报告。维护者将调查并采取适当措施，包括警告、暂时或永久封禁等。

维护者负责确保遵守本准则，并可针对任何违规行为采取行动。

## 5. 致谢

本准则改编自 [贡献者公约](https://www.contributor-covenant.org/zh-cn/version/2/0/code_of_conduct/)。

## 6. 联系方式

如有任何问题或疑虑，请联系项目维护者。
